Tailwind v4 gradient utility (verified in `dealdrop/node_modules/tailwindcss/`):
- `bg-gradient-to-b` — top-to-bottom linear gradient
- `from-orange-50` — start color (warmest stop at top of section)
- `via-background` — middle color (fades to neutral)
- `to-background` — end color (matches surrounding page; no hard edge)
- `dark:from-transparent` — suppresses the orange in dark mode (Pitfall 5)

The hex value for orange-50 is `#fff7ed` (Tailwind v4 stock); zinc-900 text on this
warmest stop yields ~14:1 contrast (passes AA easily). Dark mode: `from-transparent`
means the gradient effectively becomes background-on-background (no visual change).

FeatureCard grid bottom rhythm: existing `pb-12 sm:pb-16` on the section already
provides 48-64px bottom padding. After deleting the 'Made with love' <p> (which
itself had `mt-16` margin-top), no replacement spacer is needed — the section's own
padding handles bottom rhythm.
